## How to start

Clone this repository

```
git clone https://github.com/marutpun/Assessment-Seven-Peaks-FE.git
```

Install packages

```
# NPM
npm install

# Yarn
yarn install
```

Run local development

```
# NPM
npm run start

# Yarn
yarn run start
```

or build a development

```
# NPM
npm run build

# Yarn
yarn run build
```
